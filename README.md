# AIRecon: 基于XGBoost的恶意请求检测系统

## 📌 项目概述
AIRecon是一个基于机器学习的恶意HTTP请求检测系统，使用XGBoost算法替代传统规则引擎，实现了高精度的异常请求识别。系统通过分析Web服务器日志，实时检测SQL注入、路径遍历、命令执行等攻击行为。



## 🚀 运行指南

设置flag=1启用训练
